SAFETY IN CLOUD STORAGE

OVERVIEW

  • Users trust cloud storage providers (Dropbox, Google Drive, OneDrive, Box, etc) to keep important documents safe, secure, and available.
  • Providers cannot guarantee this
    • Downtime is inevitable.
    • Hackers or software bugs can compromise data.
    • Providers or governments may read user files.
  • Daruma provides strong new guarantees: no one provider can read, change, or delete user files.
  • Daruma stores files on several of the user's existing cloud accounts. Security and availability are guaranteed by advanced cryptography and redundancy algorithms.
  • Daruma provides the benefits of cloud storage without its inherent risks, through a standard and familiar interface.

OUR SOLUTION

ENCRYPTION

Daruma uses authenticated encryption, securing each file with a unique random key to prevent unauthorized data reads or edits.

FILE ENCODING

Using an algorithm known as Reed-Solomon encoding, Daruma efficiently splits up files for storage on cloud providers.

KEY SHARING

Using the previously academic Robust Secret Sharing algorithm, Daruma stores secret keys on cloud providers while remaining secure.
